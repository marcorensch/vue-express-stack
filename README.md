# UIKIT-VUE-EXPRESS STACK

## Project Setup
### Server
```
npm install
```
### Client
```
cd client
```
```
npm install
```

## Development
### Start Server & Client in one go:
```
npm run devStart
```
Starts Server & Client.<br>
Client will be available on <a href="127.0.0.1:8080" target="_blank">127.0.0.1:8080</a> after start.

### Start Server:
```
npm run devServer
```
### Start Client:
```
npm run devClient
```
OR:
```
cd client
```
```
npm run serve
```
